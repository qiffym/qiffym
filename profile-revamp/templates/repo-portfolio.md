## ⚡ Quick Start (≤ 5 minutes)

### Prerequisites
- Node 18+ and npm (or pnpm/yarn)

### Install & run
```bash
git clone https://github.com/qiffym/portfolio.git
cd portfolio

cp .env.example .env.local     # if env vars are needed; otherwise skip
npm install
npm run dev
```

Open http://localhost:3000.

### Build for production
```bash
npm run build && npm run start
```
